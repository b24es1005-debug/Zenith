# Features

## Astronomy discovery

- Interactive Leaflet map for finding dark-sky locations.
- Search by place name or landmark.
- Use current location or drop a manual pin.
- View weather, light pollution, astronomy, and Zenith Score intelligence.

## Meetups

- Browse community astronomy meetups.
- Create, join, leave, and delete meetups.
- Page-based pagination with persisted filter and sort state.
- Empty state guidance when no meetups match the selected filters.

## Dashboard

- Saved locations count.
- Meetups hosted.
- Meetups joined.
- Average Zenith Score.
- Upcoming astronomy events count.
- Saved location previews for quick review.

## Authentication

- Google OAuth sign-in.
- Optional GitHub OAuth sign-in.
- Prisma-backed session storage.
- Protected dashboard and meetup creation routes.
- Logout flow with session cleanup.

## Reliability and polish

- Loading states for key pages and data panels.
- Empty states for no meetups, no saved locations, and no astronomy results.
- Error states for weather, astronomy, Nominatim, and database failures.
- Accessible navigation, menu controls, and status messaging.
- SEO metadata and OpenGraph tags on major routes.

## Future Improvements

- AI based filtering of nearby location based on the zenith score, light pollution, weather.
- A texting platform for community members.
- A gallery for uploading the pictures.
- Monthly winner based on the number of meetups attended or organized.

  
## Skills acquired

- Server components and client components used intentionally.
- Prisma schema and migration management.
- Auth.js integration with OAuth and database sessions.
- External API integration with caching and rate limiting.
- Mobile-first UI and accessible interaction design.
